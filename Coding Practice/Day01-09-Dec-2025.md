## ðŸ”¹ Jenkins Pipeline â€” Checkout Git
You will learn how to **checkout a Git repository** and list files on groovy language

pipeline {      # it define the start of jenkins pipeline.
    agent any   # it tells that jenkins can run this pipeline on any available agents.
    stages {     # Collection of different steps or Stage tell the actions .
        stage('Checkout') { # Checkout is the name of the stage.
            steps {
                git branch: 'main', url: 'https://github.com/Pathnex/sample-java-app.git' #Connect to github repository and pull the main branch 
            }
        }
        stage('List Files') {
            steps {
                sh 'ls -la'   # Run a shell command to list the content.
            }
        }
    }
}
